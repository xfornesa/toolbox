#README
Just playing a bit with gulp and similar tools in order to start a project.


##Requirements
- [NodeJS](http://nodejs.org/)
- [Gulp](http://gulpjs.com/)
	`npm install gulp -g`
- [Compass](http://compass-style.org/)
	`gem update --system`
	`gem install compass`


## Installation
`npm install`

## Run
`gulp`
